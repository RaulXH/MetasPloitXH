# Installation Commands
```
git clone https://github.com/RaulXH/MetasPloitXH && cd MetasPloitXH

./MSFinstaller --install
```

# Uninstall Metasploit-framework
```
./MSFinstaller --uninstall

```
# Update Metasploit
```
./MSFinstaller --update
```
